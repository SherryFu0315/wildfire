Features

  1. For site owners:

    • Database Support:
      • Firebase
      • Wilddog.
    • Admin Function:
      • Delete comment
      • Ban users by their IP/email

  2. For all visitors (anonymous & authorized):

    • Comment (with Markdown support)
    • Mention (@username)

  3. For authorized visitors:

    • Like/dislike a comment
    • Delete own comments
    • Report inappropriate comments
    • Update user profile
      • Display name
      • Avatar
    • Personal Center:
      • Notification
